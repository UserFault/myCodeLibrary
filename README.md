# myCodeLibrary
Stack of classes for using.


1 Дата начала: 15 июля 2017 г.

2 Название: MyCodeLibrary

3 Краткое описание: Разработка моей библиотеки функций для ускорения написания программ для разных моих проектов.

4 Описание: 

4.1 Обстановка и причины создания проекта: 
Нужно ускорить написание программ. 
Поэтому нужна собственная библиотека функций в дополнение к имеющимся. 
И она должна быть хорошо документирована.

4.2 Конечный результат проекта:  
- Библиотека, пригодная для включения в мои проекты как источник функций и классов.
- Документация по коду и поиск нужной функции.

4.3 Предполагаемая ситуация для использования проекта: 
Написание нового приложения.

4.4 Предполагаемая трудоемкость проекта: 
Постепенное накопление функциональности по мере работы над проектами.

4.5 Текущее состояние проекта: Инициализация 

4.6 План работы: 
- Найти первую версию этой библиотеки и перенести ее сюда как отдельный солюшен и проект.
- Проверить возможность подключения библиотеки в мой тестовый проект-солюшен. Описать процесс подключения в документе.
- Осмотреть все мои проекты и принести из них сюда все полезные функции.
- Написать документацию по содержимому библиотеки.
